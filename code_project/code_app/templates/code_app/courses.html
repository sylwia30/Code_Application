Python

Python to łatwy do opanowania, potężny język programowania. Ma wydajne struktury danych wysokiego poziomu i proste, ale skuteczne podejście do programowania obiektowego. Łatwa składnia i dynamiczne pisanie w Pythonie wraz z jego zinterpretowaną naturą sprawiają, że jest to idealny język do pisania skryptów i szybkiego tworzenia aplikacji w wielu obszarach na większości platform.

Szczegółowy program nauczania

HTLM5

Język znaczników niezbędny dla każdego programisty. Wykorzystywany do tworzenia i prezentowania  stron i aplikacji internetowych www. Służy do opisania sposobu wyświetlania danego elementu na stronie lub do określenia sposobu wyświetlania samej strony.

Szczegółowy program nauczania

CSS

Język służący do opisu formy prezentacji (wyświetlania) stron i aplikacji www. Cascading Style Sheets czyli kaskadowe arkusze stylów to język używany do opisu układu elementów na stronie WWW. Można za jego pomocą zdefiniować wiele parametrów, takich jak rodzaj czcionek, kolor tła, wyrównanie elementów na stronie itp.

Szczegółowy program nauczania

JavaScript

JavaScript jest to skryptowy język, który umożliwia obsługę dynamicznego tworzenia treści na stronie internetowej, kontrolowanie multimediów, animację obrazów. Służy najczęściej do zapewnienia interakcji poprzez reagowanie na zdarzenia, walidacji wprowadzanych danych w formularzach lub tworzenia złożonych efektów wizualnych.

Szczegółowy program nauczania

JQuery

Najczęściej używana biblioteka programistyczna ułatwiająca tworzenie skryptów w Java Script.Pozwala osiągnąć efekty animacji na stronach i w aplikacjach, dodać dynamiczne zmiany wyświetleń, analizować aktywność użytkownika, wykonać zapytania AJAX i wiele innych...
